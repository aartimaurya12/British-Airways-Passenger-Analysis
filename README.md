# British Airways Passenger Experience Dashboard ✈️

## Project Overview
This project analyzes customer reviews of British Airways to understand passenger satisfaction and identify key service improvement areas.

The dashboard provides insights into passenger ratings across different services such as cabin staff, food, entertainment, seat comfort, and value for money. It helps identify customer pain points and overall travel experience trends.

## Objectives
- Analyze passenger satisfaction levels  
- Identify low-performing service areas  
- Compare ratings across seat types and aircrafts  
- Track rating trends over time  
- Understand customer recommendation behavior  

## Key Insights
- Overall average rating is 4.2/10  
- Only 43.2% passengers recommend British Airways  
- Food & Beverages and Entertainment received the lowest ratings  
- Economy Class passengers showed lower satisfaction levels  
- A320 aircraft received the highest number of reviews
